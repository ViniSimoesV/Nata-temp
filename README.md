# Nata

Atendendo às necessidades do Nata, Núcleo de Apoio Toxicômanos e Alcoólatras, este repositorio concentra documentação de projeto, documentos de assinatura e código-fonte da aplicação. 
A aplicação possui sistema de contas com diferentes níves de acesso, gerenciamento de projetos, gerenciamento de parceiros, gerenciamento de pontos de entrega e de coleta, gerenciamento de frota e abas de apresentações de atividades que ocorrem no Nata.
